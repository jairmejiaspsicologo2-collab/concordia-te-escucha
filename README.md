# Concordia Te Escucha

Prototipo web para diagnóstico territorial e intersectorial de salud mental en Concordia, Magdalena.

Incluye:

- Tablero con indicadores públicos SISPRO para Concordia.
- Validación local de una IPS municipal: ESE Hospital Local de Concordia.
- Carga de archivos CSV anónimos para el periodo 2020-2026.
- Validación básica para evitar campos identificables.
- Vista territorial con Google Maps.
- Matriz de fuentes oficiales e intersectoriales.

Abre la app desde `outputs/concordia-te-escucha/index.html` o usando un servidor local en esa carpeta.
